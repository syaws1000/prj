### **언어 (Language)**

* **프런트엔드 (Front-end)**
    * **HTML5**: 웹 페이지의 구조와 콘텐츠를 정의하는 마크업 언어입니다.
    * **CSS3**: HTML 요소들의 시각적 스타일을 꾸미는 스타일시트 언어입니다.
    * **JavaScript**: 웹 페이지에 동적인 기능을 추가하는 스크립트 언어입니다.

* **백엔드 (Back-end)**
    * **Java17**: 안정성이 높아 엔터프라이즈급 웹 애플리케이션 개발에 널리 사용되는 객체 지향 프로그래밍 언어입니다.
    * **Python**: 간결하고 배우기 쉬워 웹 개발, 데이터 분석, 인공지능 등 다양한 분야에서 활용되는 언어입니다.
    * **JSP (JavaServer Pages)**: HTML 내에 자바 코드를 삽입하여 동적인 웹 페이지를 생성하는 서버 사이드 스크립트 언어입니다.

### **프레임워크 (Framework)**

* **Spring**: 자바 기반의 웹 애플리케이션 개발을 위한 포괄적인 기능을 제공하는 프레임워크입니다.
* **SpringBoot**: Spring을 더 쉽고 빠르게 개발할 수 있도록 도와주는 Spring의 서브 프로젝트입니다.
* **Bootstrap**: 반응형 웹 디자인을 쉽게 구현할 수 있도록 도와주는 프런트엔드 프레임워크입니다.
* **Flask**: 파이썬으로 웹 애플리케이션을 만들 수 있게 도와주는 마이크로 웹 프레임워크입니다.
* **Vue.js**: 사용자 인터페이스를 만들기 위한 프로그레시브 자바스크립트 프레임워크입니다.
* **MyBatis**: 자바에서 데이터베이스 프로그래밍을 더 쉽게 할 수 있도록 도와주는 SQL 매퍼(Mapper) 프레임워크입니다.

### **툴 (Tool)**

* **Eclipse (STS)**: 자바 기반의 통합 개발 환경(IDE)이며, STS는 Spring 개발에 특화된 버전입니다.
* **Visual Studio Code**: 마이크로소프트에서 개발한 가볍고 확장성이 뛰어난 코드 에디터입니다.
* **eXERD**: 데이터베이스 모델링을 위한 툴입니다.
* **SVN (Subversion)**: 소스 코드의 버전 관리를 위한 형상 관리 툴입니다.

### **데이터베이스 (DB)**

* **Oracle 11g**: 대용량 데이터 처리에 강점을 가진 관계형 데이터베이스 관리 시스템(RDBMS)입니다.

### **웹 서버 (Web Server)**

* **Tomcat**: JSP와 서블릿을 실행하기 위한 웹 애플리케이션 서버(WAS)입니다.

### **운영체제 (OS)**

* **Window**: 개발 환경으로 사용되는 운영체제입니다.
