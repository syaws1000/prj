## 4. 주요 개념/핵심 키워드를 최고의 암기법에 의한 제시 (Gagné 4단계)

### 🔑 \[키워드 + 암기법]

| 개념                  | 설명               | 기억법                    |
| ------------------- | ---------------- | ---------------------- |
| `<jsp:useBean>`     | 자바 객체를 JSP에 불러오기 | **Bean(콩): JSP에 콩 심기** |
| `<jsp:setProperty>` | 폼 데이터 객체에 담기     | **Set(설정) 폼값 저장**      |
| `$(선택자).on()`       | jQuery 이벤트 연결    | **On! 스위치 켜기**         |

> 🧠 **손가락 암기법**: 콩(Bean) → 설정(Set) → 스위치(On!)
> (손가락 3개로 외워보세요!)

## 5. 학습 목표별 탐구형 학습 (Gagné 5단계)

### 🧐 \[탐구형 문제제시]

* **문제**: 회원가입 폼에서 이름, 이메일을 입력하고 ‘가입’ 버튼을 누르면, JSP로 데이터를 전송하고 jQuery로 알림을 띄우세요.
* **가이드**:

  1. `form` 태그 작성
  2. `useBean`으로 폼 데이터 처리
  3. jQuery로 버튼 클릭 이벤트 설정

> 💬 **힌트**: `$("#submitBtn").on("click", function(){ alert("가입 완료!"); });`
