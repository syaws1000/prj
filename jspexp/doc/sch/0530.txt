## 4. 주요 개념/핵심 키워드를 최고의 암기법에 의한 제시 (Present Content, C - Confidence)

**\[핵심 키워드]**

| 개념                                 | 암기법 (기억 도우미)               |
| :--------------------------------- | :------------------------- |
| `<jsp:useBean>`                    | "자바 오브젝트를 JSP로 '소환'하는 마법진" |
| `<jsp:setProperty>`                | "오브젝트에 값 세팅하는 '주문'"        |
| `<jsp:getProperty>`                | "오브젝트 값을 '꺼내는' 도구"         |
| jQuery `$(선택자).이벤트()`              | "선택하고, 명령하고, 즐긴다!"         |
| jQuery `text()`, `html()`, `val()` | "문자, 태그, 값 조작 삼총사"         |

**📝 암기 팁**

* **UseBean**: 객체 소환
* **SetProperty**: 값 넣기
* **GetProperty**: 값 꺼내기
* **\$()**: 잡아라(선택자)
* **이벤트 함수**: 움직여라

**🔥 그림 + 짤 활용**

* 액션태그: "JSP 도구 3총사"
* jQuery: "웹페이지의 놀이터(jQuery Park)"
