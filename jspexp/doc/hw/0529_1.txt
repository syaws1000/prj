### 객관식 문제 15문제:

1. **JSP에서 세션 범위(session scope)에 저장되는 데이터는 언제까지 유효한가요?**

   * A. 페이지가 새로 로드될 때
   * B. 한 HTTP 요청이 끝날 때
   * C. 클라이언트가 웹 애플리케이션을 종료할 때까지
   * D. 웹 애플리케이션 서버가 재시작될 때까지

2. **다음 중 `response.sendRedirect()`를 사용한 후의 결과로 올바른 것은 무엇인가요?**

   * B. 클라이언트의 URL이 변경되고 새로운 요청이 서버로 전송된다.
   * C. 클라이언트의 요청 데이터는 그대로 유지된다.
   * D. URL에 포함된 모든 파라미터가 자동으로 제거된다.

3. **JSP에서 `forward`를 사용할 때의 특징으로 올바른 것은 무엇인가요?**

   * A. 클라이언트의 URL이 변경된다.
   * B. 요청과 응답 객체를 그대로 유지한 채 다른 JSP 페이지로 이동한다.
   * C. 세션 정보는 사라진다.
   * D. 요청 정보를 다른 페이지로 전달할 수 없다.

4. **다음 중 `session scope`에 저장된 값이 사라지는 시점은 언제인가요?**

   * A. 웹 애플리케이션 서버가 종료될 때
   * B. 페이지가 새로 로드될 때
   * C. 세션 타임아웃이 발생할 때

   * A. 클라이언트에서 데이터를 가져오기 위한 태그

   * C. HTML 폼 데이터를 서버로 전송하는 기능을 제공한다.
   * D. 페이지 내에서 자바스크립트 코드만을 실행할 수 있게 한다.

6. **JSP에서 `c:if` 태그를 사용하여 조건문을 처리하는 데 필요한 속성은 무엇인가요?**

   * D. 서버에서 클라이언트로 데이터를 보내는 것

* A. 사용자가 요청한 페이지에서만 유효하다.

* C. 세션 값을 초기화하기 위해
* D. 조건문을 처리하기 위해

* A. 서버에서 다른 페이지로 자동 전환된다.

* C. `redirect`는 URL을 변경하고, 새 요청을 클라이언트에서 서버로 보낸다.
* D. `redirect`는 서버 내에서 페이지 이동을 하며, `forward`는 클라이언트의 URL을 변경하지 않는다.
