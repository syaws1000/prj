# 프로젝트 java 예시 프로그램
1. **학생 관리 시스템:** 콘솔 기반의 학생 관리 시스템을 Java로 개발하여, 학생의 정보를 추가하고, 수정하고, 삭제할 수 있도록 만들기.

2. **도서관 도서 관리:** 도서의 제목과 저자를 기준으로 도서를 대출하고 반납하며, 도서 목록을 검색할 수 있는 Java 프로그램 구현.

3. **은행 계좌 시뮬레이션:** Java로 은행 계좌 프로그램을 만들어 입금, 출금, 잔액 조회 기능을 구현.


#### 1. **도서 관리 시스템 구축**

* **문제:** 도서관에서 새로운 도서 관리 시스템을 도입하려고 합니다. 도서 정보를 저장할 테이블을 설계하고, 도서 등록, 조회, 수정, 삭제 기능을 구현하세요.
* **요구사항:** 도서 ID, 제목, 저자, 출판사, 출판 연도, 장르 등의 정보를 관리할 수 있어야 합니다.

#### 2. **학생 정보 관리 시스템**

* **문제:** 학교에서 학생들의 정보를 관리하기 위한 시스템을 구축하려고 합니다. 학생 정보를 저장할 테이블을 설계하고, 학생 등록, 조회, 수정, 삭제 기능을 구현하세요.
* **요구사항:** 학생 ID, 이름, 나이, 학년, 전공 등의 정보를 관리할 수 있어야 합니다.

#### 3. **직원 급여 관리 시스템**

* **문제:** 기업에서 직원들의 급여 정보를 관리하기 위한 시스템을 구축하려고 합니다. 직원 급여 정보를 저장할 테이블을 설계하고, 급여 등록, 조회, 수정, 삭제 기능을 구현하세요.
* **요구사항:** 직원 ID, 이름, 부서, 직책, 급여 등의 정보를 관리할 수 있어야 합니다.

#### 4. **상품 재고 관리 시스템**

* **문제:** 상점에서 상품의 재고 정보를 관리하기 위한 시스템을 구축하려고 합니다. 상품 재고 정보를 저장할 테이블을 설계하고, 재고 등록, 조회, 수정, 삭제 기능을 구현하세요.
* **요구사항:** 상품 ID, 이름, 가격, 재고 수량, 공급업체 등의 정보를 관리할 수 있어야 합니다.

#### 5. **고객 주문 관리 시스템**

* **문제:** 온라인 쇼핑몰에서 고객의 주문 정보를 관리하기 위한 시스템을 구축하려고 합니다. 주문 정보를 저장할 테이블을 설계하고, 주문 등록, 조회, 수정, 삭제 기능을 구현하세요.
* **요구사항:** 주문 ID, 고객 ID, 상품 ID, 주문 수량, 주문 일자 등의 정보를 관리할 수 있어야 합니다.
