아래는 주어진 HTML 및 CSS 학습 내용을 기준으로 단계별 학습 완료 체크리스트입니다:

### **1. HTML 기본 개념**
- [ ] HTML의 기본 구조(<!DOCTYPE html>, `<html>`, `<head>`, `<body>`) 이해
- [ ] `h1`~`h6`, `p`, `hr`, `br` 태그 사용
- [ ] 텍스트 관련 태그인 `strong`, `em`, `mark` 사용
- [ ] `ul`, `ol`, `li`를 사용하여 리스트 구성
- [ ] 이미지 삽입 태그 `<img src="">` 사용

### **2. HTML 2단계**
- [ ] `form` 요소 사용하여 사용자 입력 처리
- [ ] `iframe` 요소 사용하여 다른 페이지 삽입
- [ ] HTML 기본 속성 사용 (예: `id`, `class`, `alt`, `title`, `target`)

### **3. CSS 기초 문법**
- [ ] CSS 선택자(`{속성:값}`) 사용
- [ ] 클래스 선택자(`.class`) 및 아이디 선택자(`#id`) 사용
- [ ] 가상클래스 선택자 (`:hover`, `:focus`, `:active`) 사용
- [ ] 텍스트 스타일링 (글꼴, 크기, 색상 등) 적용
- [ ] 텍스트 시각적 꾸미기 (볼드, 이탤릭, 밑줄 등) 처리
- [ ] 박스 모델 이해 (패딩, 마진, 보더 등)
- [ ] 레이아웃 구성 (블록 요소, 인라인 요소 등)
- [ ] 배경색 및 배경 이미지 설정
- [ ] 배경 디자인 효과 적용 (예: `background-image`, `background-repeat` 등)

### **4. CSS 중급**
- [ ] CSS 속성 적용 및 이해 (예: `width`, `height`, `border`, `font-size` 등)
- [ ] 요소 배치 제어 (예: `float`, `position`, `display`,)
