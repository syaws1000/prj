
## ✅ 주관식 개념문제 10문제

1. `TO_NUMBER()` 함수는 어떤 상황에서 사용되며, 주의할 점은 무엇인가요?

2. `TO_CHAR(SYSDATE, 'YYYY/MM/DD')` 구문의 결과는 어떤 형식의 문자열인가요? 각각의 포맷 요소가 의미하는 바를 설명하세요.

3. `TO_CHAR()` 함수를 이용하여 급여 정보를 천 단위 콤마(,)와 함께 출력하려면 어떤 형식을 지정해야 하나요?

4. `TO_DATE()` 함수는 어떤 상황에서 주로 사용되며, 데이터의 어떤 문제를 해결해줄 수 있나요?

5. `SYSDATE`를 기준으로 현재 시간(24시간제), 분, 초를 문자열로 출력하는 SQL 문을 작성해 보세요.

6. Oracle에서 `SEQUENCE` 객체의 주요 속성값 3가지를 설명하고, 각각 어떤 역할을 하는지 서술하세요.

7. `SEQ_EXAMPLE.NEXTVAL`과 `SEQ_EXAMPLE.CURRVAL`의 차이점은 무엇인가요?

8. `DAO` 클래스에서 SQL을 실행하기 위한 `PreparedStatement` 객체는 어떤 방식으로 생성되고, 어떤 장점이 있나요?

9. Java에서 다중 행, 단일 열의 결과를 처리하기 위해 어떤 컬렉션을 사용하며, 그 이유는 무엇인가요?

10. 아래 SQL 문을 Java DAO 메서드에서 사용할 때, 조건문에 동적으로 사용자 입력값을 넣기 위해 어떤 처리를 해야 하나요?

```sql
SELECT * FROM EMP WHERE ENAME LIKE ? AND JOB LIKE ?
```

## ✅ 이야기가 있는 확인 연습문제 5문제

### 1. **"문자열 급여 문제"**

회계팀의 김과장은 부서 직원들의 급여 데이터를 Excel로 정리하려고 합니다. 그런데 급여는 모두 문자열 '3,500'과 같은 형식으로 입력되어 있어 계산이 불가능했습니다.
**문제:**
이 급여들을 숫자형으로 변환하여 3500 이상인 직원만 조회하는 SQL 문을 작성하세요.

---

### 2. **"입사일 로그 시스템"**

개발팀은 사원들의 입사일 데이터를 로그에 남기기 위해 `YYYY-MM-DD HH24:MI:SS` 형식으로 출력하려고 합니다.
**문제:**
사원번호, 사원명, 입사일을 해당 형식으로 출력하는 SQL 문을 작성하세요.

---

### 3. **"자동 회원번호 생성기"**

스타트업 '다함께회사'는 회원가입 시 회원번호를 자동으로 1001번부터 부여하고자 합니다.
**문제:**
이 요구사항을 반영한 `SEQUENCE` 생성 구문을 작성하세요.

---

### 4. **"DAO 처리 실수"**

신입 개발자 민수는 아래와 같이 DAO 메서드를 작성했지만 데이터가 조회되지 않았습니다.

```java
String sql = "SELECT * FROM EMP WHERE ENAME = ? AND JOB = ?";
pstmt.setString(1, sch.getEname());
pstmt.setString(2, sch.getJob());
```

하지만 사용자가 입력한 검색 조건은 `이름에 A가 포함`되고 `직책에 MAN이 포함`된 데이터입니다.
**문제:**
해당 조건에 맞도록 SQL과 `setString` 구문을 수정해 보세요.

---

### 5. **"다중 행 이름 수집기"**

부장님은 부서번호가 30번인 모든 직원 이름을 리스트로 받아보고 싶어 합니다.
**문제:**
이 요구를 만족시키기 위한 Java DAO 메서드를 작성할 때, 반환 타입과 SQL 구문을 기술하세요.
