


#### 【JavaScript 조건문/반복문/배열 관련】

**1.** JavaScript에서 조건이 참일 때만 특정 코드를 실행하는 구문은 무엇인가?  
(A) for문 (B) if문 (C) while문 (D) switch문  

**2.** `prompt()` 함수는 어떤 용도로 사용되는가?  
(A) 조건 판단 (B) 사용자 입력 받기 (C) 배열 생성 (D) 반복문 종료  

**3.** JavaScript의 if-else문에서 else문은 언제 실행되는가?  
(A) 조건이 참일 때 (B) 조건이 거짓일 때 (C) 무조건 실행 (D) 오류 발생 시  

**4.** 숫자가 홀수인지 짝수인지를 판별할 때 사용하는 연산자는?  
(A) % (B) * (C) + (D) /  

**5.** `for(let i=0; i<5; i++)`는 총 몇 번 반복하는가?  
(A) 4번 (B) 5번 (C) 6번 (D) 3번  

**6.** `while(true){}` 반복문을 반드시 중단하려면 어떤 키워드를 사용해야 하는가?  
(A) continue (B) stop (C) break (D) exit  

**7.** `document.body.innerHTML += "내용";` 구문은 어떤 의미를 가지는가?  
(A) 기존 내용 삭제 후 출력 (B) 새 페이지로 이동  
(C) 기존 내용에 새로운 내용을 추가 (D) 입력값을 저장  

**8.** 배열에서 요소를 추가할 때 사용하는 메서드는 무엇인가?  
(A) pop() (B) push() (C) shift() (D) splice()  

**9.** 배열 `['a','b','c']`에 대해 `pop()`을 호출하면 결과는?  
(A) 'a'가 제거된다 (B) 'b'가 제거된다  
(C) 'c'가 제거된다 (D) 아무 것도 제거되지 않는다  

**10.** `splice(1,2)`를 배열 `[1,2,3,4]`에 적용하면 어떤 변화가 발생하는가?  
(A) 첫 번째 요소 삭제 (B) 두 번째와 세 번째 요소 삭제  
(C) 네 번째 요소만 삭제 (D) 삭제 없이 추가만 가능

#### 【Oracle SQL CASE WHEN 관련】

**11.** Oracle SQL의 `CASE WHEN` 구문은 주로 어디에서 사용되는가?  
(A) 데이터 입력 (B) 데이터 삭제 (C) 조건에 따른 값 반환 (D) 테이블 생성  

**12.** "사원번호가 7654 이하면 홍팀"처럼 범위를 기준으로 조건 분기를 하려면 어떤 CASE 유형을 써야 하는가?  
(A) 단순 CASE (B) 검색 CASE(Searched CASE) (C) GROUP BY (D) JOIN  

**13.** 아래 중 CASE 구문의 ELSE 절 역할은 무엇인가?  
(A) WHERE 절을 대체한다  
(B) WHEN 조건이 모두 틀릴 때 기본 결과를 지정한다  
(C) 조건을 추가한다  
(D) 데이터를 삭제한다  

**14.** `CASE 컬럼명 WHEN 값1 THEN 결과1 ELSE 결과2 END` 형태는 어떤 CASE인가?  
(A) 검색 CASE (B) 단순 CASE (C) 복합 CASE (D) 무조건 CASE  

**15.** 사원 테이블(EMP)에서 부서 번호(DEPTNO)에 따라 '인사부', '회계부', 'IT부'를 구분하는 쿼리에 적합한 구문은?  
(A) UPDATE (B) CASE WHEN (C) GROUP BY (D) ORDER BY
