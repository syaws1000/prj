
### ✅ 자바(Java) 응용 실습 문제 (20문제)

1. **🎰 행운 번호 추첨기**  
   - 1~100 사이의 숫자 중 랜덤으로 추첨하고, 7의 배수면 "럭키!" 출력하세요.

2. **🎓 합격자 판별기**  
   - 사용자에게 점수를 입력받아 60점 이상이면 "합격", 아니면 "불합격"을 출력하세요.

3. **🍰 케이크 조각 나누기**  
   - 케이크 조각 수와 친구 수를 입력받아 1인당 몇 조각인지, 몇 조각 남는지 출력하세요.

4. **📮 우편번호 자동 분류기**  
   - 우편번호 앞자리를 기준으로 "서울", "부산", "기타"로 분류하여 출력하세요.

5. **🧮 짝수/홀수 구분기**  
   - 1~20까지 출력하면서 짝수는 "(짝)", 홀수는 "(홀)"로 출력하세요.

6. **🎳 볼링 점수 누적기**  
   - 사용자로부터 10번 점수를 입력받고 총점과 평균을 출력하세요.

7. **📋 투표 결과 집계기**  
   - 후보 3명의 득표 수를 입력받아 가장 높은 득표자를 출력하세요.

8. **📅 월별 일정 확인기**  
   - 월(1~12)을 입력받아 해당 월의 일 수를 출력하세요. (윤년 고려X)

9. **🧻 화장실 줄 서기 시뮬레이터**  
   - 사람 수를 입력받아, 3번째 사람마다 "참다 못 참음"을 출력하세요.

10. **🎨 색상 맞추기 게임**  
   - 사용자에게 원하는 색상("red", "green", "blue")을 입력받고 정답과 비교하세요.

11. **🚂 기차표 좌석 분배기**  
   - 40개의 좌석이 있다고 가정하고, 사용자가 입력한 인원 수만큼 좌석 번호를 할당하세요.

12. **🍔 햄버거 주문 자동계산기**  
   - 햄버거 종류와 수량을 입력받아 총 주문 금액을 출력하세요.

13. **🎯 점수에 따른 학점 출력기**  
   - 점수를 입력받아 A~F 학점을 출력하세요 (switch문 사용)

14. **🛵 배달 거리 계산기**  
   - 거리를 입력받고 3km 이하는 기본요금, 그 이상은 거리당 추가요금을 계산하세요.

15. **🧹 청소 로봇 경로 추적기**  
   - for문을 이용해 청소로봇이 1~10까지 청소하며, 5에서 "충전 중"을 출력하세요.

16. **🎲 주사위 2개 게임**  
   - 주사위 2개를 굴려 합이 짝수면 "승리", 홀수면 "패배" 출력하세요.

17. **⛳ 미니 골프 점수판**  
   - 5홀 점수를 입력받아 평균과 최고 점수를 출력하세요.

18. **🕹️ 점프 게임 판정기**  
   - 입력받은 점프 거리 10 이상이면 "성공", 아니면 "실패" 출력하세요.

19. **📦 박스 무게 측정기**  
   - 5개의 무게를 배열에 저장하고, 가장 무거운 박스를 출력하세요.

20. **🎮 게임 랭킹 보드 만들기**  
   - 유저 5명의 점수를 입력받고, 높은 순서대로 출력하세요.

### ✅ HTML/CSS 응용 실습 문제 (20문제)

21. **🎨 나의 색상 선택기**  
   - `<input type="color">`를 사용해 배경색을 바꾸는 색상 선택기를 만들기

22. **🍽️ 오늘의 식단표 꾸미기**  
   - `table`로 식단표를 만들고 `:hover` 시 행 색상 변경

23. **🎫 티켓 스타일 카드**  
   - `div`로 영화 티켓처럼 꾸미고, `border-radius`로 모서리 둥글게 만들기

24. **💌 메일 입력창 강조하기**  
   - `input[type=email]`에 `focus` 시 테두리 색 변경하기

25. **📢 스크롤에도 보이는 고정 공지창**  
   - `position: fixed`를 이용해 오른쪽 아래 공지창 고정

26. **📚 책 소개 카드 꾸미기**  
   - 책 정보를 담은 `div` 3개를 `inline-block`으로 가로 정렬

27. **🖼️ 배경 이미지 꾸미기**  
   - 이미지가 반복되지 않도록 하고, 가운데 정렬되게 설정

28. **🎡 테마파크 소개 섹션 꾸미기**  
   - `section`에 `background`, `padding`, `box-shadow` 등 스타일 추가

29. **🕹️ 버튼 클릭 시 스타일 바뀌기**  
   - `button:active`로 클릭 중 스타일 변경

30. **📦 박스 모델 시각화**  
   - margin, padding, border를 각각 다른 색으로 표현

31. **🎯 마우스 오버 효과 주기**  
   - 여러 `div`에 `:hover`로 색상 전환 효과 만들기

32. **📂 파일 업로드 디자인**  
   - 파일 input에 `label`을 이용한 업로드 버튼 디자인하기

33. **🕶️ 제품 광고 카드**  
   - 상품 정보를 보여주는 카드에 그림자(`box-shadow`) 효과 추가

34. **📦 카드 쌓기 z-index 사용**  
   - 여러 `div`를 겹쳐 배치하고 z-index로 순서 조정

35. **📋 폼 제출 스타일**  
   - `form` 안의 input, button 정렬 및 스타일 지정

36. **🔑 로그인 박스 중앙 배치**  
   - 로그인 폼을 화면 중앙에 정렬 (absolute or flex 사용)

37. **🎉 마우스 오버로 등장 애니메이션**  
   - `transition`을 사용해 `hover` 시 크기 변화 애니메이션 적용

38. **🍀 lucky draw 박스 스타일링**  
   - lucky draw 버튼 클릭 시 박스 색상 변경되게 만들기 (class 변경 방식)

39. **🧁 디저트 메뉴판**  
   - `ul > li`로 메뉴 구성 후, 마우스 오버 시 확대 효과 적용

40. **🧲 마우스 따라다니는 배너**  
   - `position: fixed`와 `z-index`를 조합해 마우스 근처에 고정 배너 만들기
