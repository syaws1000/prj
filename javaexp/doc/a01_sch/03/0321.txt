## ⭐ **4. 주요 개념/핵심 키워드를 최고의 암기법에 의해 제시 (Confidence)**  

| 개념      | 설명                         | 암기법                           |
|-----------|----------------------------|----------------------------------|
| 변수      | 데이터를 저장하는 공간       | **변(경할 수 있는) 수(값)** = 바뀌는 값 저장 공간! |

👉 **변수 = 주소에 이름 붙이기, 타입은 그릇 크기 정하기, 형변환은 그릇 옮기기!**

## 🧩 **5. 학습 목표별 탐구형 학습 (Satisfaction)**  

### ✔️ 탐구 미션 1 — 변수 선언과 사용 실습  
```java
int age = 25;  
double height = 175.5;  
char grade = 'A';  문자한자 
boolean pass = true;  
String str = "홍길동"; 문자의배열
System.out.println(age + ", " + height + ", " + grade + ", " + pass);
```
💡 _어떤 데이터가 어떤 자료형에 맞을까? 직접 실습 후 토론!_  

---

### ✔️ 탐구 미션 2 — 자동 형변환 탐색  
```java
int i = 10;  
double d = i; // 자동 형변환  
System.out.println("자동 변환 결과 : " + d);
```
🙋‍♀️ **질문:**  
> "왜 자동으로 변환이 됐을까요? 큰 컵에 작은 컵 물 부어 넣듯 자연스러운 현상!"  

---

### ✔️ 탐구 미션 3 — 강제 형변환 실험  
```java
double dd = 123.456;  
int ii = (int) dd;  // 강제 형변환  
System.out.println("강제 변환 결과 : " + ii);
```
💥 **주의 질문:**  
> "왜 소수점이 사라졌을까요? 강제 형변환 시 주의사항은 무엇일까요?"
