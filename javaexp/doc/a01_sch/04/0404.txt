## 💻 자바 반복문 학습 교안 (Gagné × ARCS 통합 설계)  
> 다양한 능력의 학습자를 위한 단계별 자기 주도 학습 + 실무 적용 + 유쾌한 참여형 수업  

---

### 1. 🎯 주의 집중 끌기 및 학습 연결성 제공  
**[ARCS: Attention & Relevance]**

- 🧠 오프닝 퀴즈:  
  “Q. 세상에서 가장 많이 도는 것은?  
  1) 지구 2) 팬 3) 자바의 for문 4) 삼겹살 뒤집개”  
  👉 정답: **3번 for문!**  
  *(웃음 유도 + 주제 집중)*

- 💡 동기 부여 영상:  
  반복문이 없으면 어떤 문제가 생기는지 보여주는 10초 애니메이션 (ex. 사람이 직접 100줄 코딩!)

- 💬 “여러분, 같은 코드를 100번 복사해서 붙여 넣으시겠습니까? … 아니죠, **코드는 반복하지 말고, 반복문을 써야죠!**”

---

### 2. 📌 학습 목표 명시  
**[Gagné 2단계 + ARCS: Confidence]**

- 자바의 반복문(for, while, do-while)의 구조와 차이점을 설명할 수 있다.  
- 반복문을 활용한 반복 처리 문제를 해결할 수 있다.  
- 실무에서 반복문을 활용한 간단한 자동화 코드를 구현할 수 있다.

---

### 3. 🔁 선수 학습 상기 및 실생활 연결하기  
**[Gagné 3단계 + ARCS: Relevance]**

- ⚙️ 선수 학습: 조건문(if, switch)  
  → “조건이 맞을 때만 실행 = 반복 전에 반드시 필요했던 개념!”

- 🛒 실생활 연결:  
  - 마트에서 가격표를 100개 붙일 때, 사람이 직접 하나하나?  
  - 루프를 쓰면 컴퓨터가 “착착착~” 반복 처리!

---

### 4. 🧠 주요 개념/핵심 키워드 제시 + 최고의 암기법  
**[Gagné 4단계]**

| 반복문 종류 | 구조 (암기법 포함) | 특이점 |
|------------|--------------------|--------|
| `for`문 | `for(초기화; 조건; 증감)` → ‘**초·조·증**’ (초콜릿 좋아하는 조증 친구!) | 정해진 횟수 반복에 적합 |
| `while`문 | `while(조건)` → ‘**조건이 참일 동안 돈다**’ | 조건만 보고 반복 |
| `do-while`문 | `do { ... } while(조건);` → ‘**먼저 하고 조건 본다**’ | 최소 1회는 실행 |

✅ **메모리 팁**:  
- `for`는 “정해진 횟수”  
- `while`은 “조건만 보고 돌기”  
- `do-while`은 “일단 하고 생각”

---

### 5. 🔍 학습 목표별 탐구형 학습  
**[Gagné 5단계 + ARCS: Relevance + Confidence]**

- 💡 *탐구 과제 1:* for문으로 1~10까지 출력하기  
- 💡 *탐구 과제 2:* while문으로 짝수만 출력  
- 💡 *탐구 과제 3:* do-while문으로 최소 1회 실행 확인하기

**👉 탐구 가이드 카드 제공:**  
각 문법별 구조, 실행 순서, 유의사항을 체크리스트로 제공

---

### 6. 🛠️ 실습 활동 및 협력 학습 유도  
**[Gagné 6단계 + ARCS: Satisfaction]**

- 🤝 **짝 활동**:  
  - "내 친구는 어떤 반복문이 더 익숙할까?" 짝과 문법 서로 설명해보기  
  - 서로 다른 반복문으로 같은 문제 해결 (예: 1~100까지 합 구하기)

- 🧪 **도전 과제**:  
  "숫자 맞히기 게임 만들기" (무한 반복 → while문 사용 → 맞히면 break!)

---

### 7. 🪞 단계별 피드백 제공과 자기 평가 기회  
**[Gagné 7단계]**

- ✅ 퀴즈 타임:  
  - 빈칸 채우기 / 코드 순서 정렬 / 참·거짓 퀴즈  
- 🔄 자기 점검표:  
  - 나는 반복문 구조를 정확히 기억하는가?  
  - 각 반복문을 어떤 상황에 써야 할지 구분할 수 있는가?

---

### 8. 🧪 수행 평가 및 피드백 제공  
**[Gagné 8단계]**

- 🎯 **미니 프로젝트**:  
  - “카페 자동 주문 처리 프로그램” 만들기  
    → 반복문을 사용해서 음료 주문 5번 받고, 총합 계산

- 📬 **교사 피드백 제공**:  
  - 반복문 구조 잘 썼는가?  
  - break와 continue 정확히 사용했는가?
