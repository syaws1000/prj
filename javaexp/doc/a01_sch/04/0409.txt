### ✅ 4. 주요 개념/핵심 키워드를 최고의 암기법으로 제시 (Gagné 4단계)

**📌 핵심 키워드 (기억법 포함):**  
| 개념 | 설명 | 기억법 |
|------|------|--------|
| DBMS | Database Management System | "데이터를 관리하는 집사(DB-MS)" |
| Oracle | 대표적인 DBMS 제품 | "오라클 = 세계적인 데이터 요리사" |
| SQL | Structured Query Language | "S!Q!L! → 셀렉하고, 끄집어내고, 넣는다!" |
| SELECT | 데이터 조회 구문 | “SELECT = 뽑아와!” |
| TABLE | 데이터 저장 구조 | “표(table)에 데이터 한 줄씩 차곡차곡~” |
