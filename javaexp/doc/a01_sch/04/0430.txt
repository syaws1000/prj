- 🧠 **연결성**:  
  "이 비밀 도구가 바로 오늘 배울 ‘**함수(function)**’예요!  
  함수를 알면 복붙 천국, 시간 절약의 달인이 됩니다!"

- 🎯 **주의 집중 활동**:  
  - "2+2 계산을 100번 시키는 코드를 직접 보며 '짜증 지수'를 끌어올리고,

## 2. 학습 목표 (Gagné 2 + ARCS Relevance)

> ✨ 오늘 우리가 스스로 해내야 할 목표입니다!

- 🔹 함수(Function)의 개념을 이해하고 직접 작성할 수 있다.
- 🔹 매개변수와 반환값의 의미를 설명하고 활용할 수 있다.
- 🔹 주요 내장 함수(`alert()`, `prompt()`, `parseInt()`, `Math.random()`)를 활용할 수 있다.
- 🔹 문제 해결에 함수와 내장함수를 적절히 적용할 수 있다.

## 4. 주요 개념/핵심 키워드를 최고의 암기법에 의한 제시 (Gagné 4)

> 🧠 암기코드 : "함수는 **'미리 써놓고', '필요할 때 부르는' 마법주문**"

- 📚 핵심 키워드 정리:
| 키워드 | 기억 코드 | 예시 |
|:---|:---|:---|
| `function` | "공장 세팅" | function sayHi(){ alert('Hi!') } |
| `parameter` | "재료 가져오기" | (name) |
| `return` | "완성품 돌려주기" | return name + '님, 안녕하세요!' |
| `alert()` | "알람 울리기" | alert('경고!') |
| `prompt()` | "질문 던지기" | prompt('이름은?') |
| `parseInt()` | "숫자 변환 마법" | parseInt('10') |
| `Math.random()` | "무작위 숫자 뽑기" | Math.random() |

- 🔥 **기억 도전 미션**:  
  각 키워드를 **1줄 문장으로 스토리처럼** 기억하기  
  (예: "공장 세팅하고, 재료 가져와서, 완성품을 돌려준다!")

## 5. 학습 목표별 탐구형 학습 (Gagné 5)

- 🎯 [탐구 활동 1]  
  - 스스로 `function`을 사용하여 "나를 소개하는" 함수를 만들어보기
- 🎯 [탐구 활동 2]  
  - `prompt()`로 나이 물어보고 `parseInt()`로 변환 후, 성인/미성년자 판별
- 🎯 [탐구 활동 3]  
  - `Math.random()`을 사용해 "운세 뽑기" 프로그램 만들기
