## 🔍 5. [학습 목표별 탐구형 학습 진행]
단계별 프로젝트 활동:

| 단계 | 주요 활동 |
|------|-----------|
| 1단계 | **좋아하는 사이트 선정 및 기능 분석** |
| 2단계 | **와이어프레임 작성, 화면 설계** |
| 3단계 | **HTML/CSS로 메인화면 및 핵심 기능 구현** |
| 4단계 | **JavaScript로 기능 추가 (예: 버튼, 팝업 등)** |
| 5단계 | **DB 설계 및 SQL 작성** |
| 6단계 | **JSP/Java로 DB 연동 기능 구현** |
| 7단계 | **Spring으로 구조화된 백엔드 설계** |
| 8단계 | **AI/빅데이터 기술 연동 (선택)** |

> 🤹 프로젝트는 팀 or 개인 모두 가능! 자율성과 책임을 동시에!
