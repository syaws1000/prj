## **4. 주요 개념/핵심 키워드 제시 (최고의 암기법 포함)** (Gagné 4단계 + ARCS Confidence)

> 🧠 **기억법으로 개념을 싹!**  

| 키워드 | 설명 | 기억법 |
|--------|------|--------|
| `display` | 요소의 배치 방법 | “디자인의 문을 여는 키!” |
| `position` | 위치 지정 (static, relative, absolute 등) | “포지션! 축구 포지션만 있는 게 아니죠~” |
| `margin/padding` | 바깥/안쪽 여백 | “마파(MaPa)도 CSS 가족이에요” |
| `border` | 테두리 | “보더(brother)는 테두리를 지켜요” |
| `form-style` | input, button 디자인 | “폼 나게 꾸며야 진짜 폼!” |
| `table-style` | 선 정리, 배경색 지정 | “표(table)는 표정이 중요하죠!” |

## **5. 학습 목표별 탐구형 학습** (Gagné 5단계 + ARCS Confidence)

> ✍️ 직접 탐색하고 만들어보는 시간!

- **탐구 과제 1**:  
  ➤ `display: flex`와 `position`을 각각 사용하여 박스를 원하는 위치에 배치해보세요.

- **탐구 과제 2**:  
  ➤ 로그인 form을 만들어 margin, padding, border-radius 등을 조절해보기.

- **탐구 과제 3**:  
  ➤ 테이블을 만들어 `border-collapse`, `nth-child`를 활용하여 꾸며보기.

- 👨‍🔧 **힌트 제공**:  
  - “Flex는 자석, Position은 GPS다!”
